## Problem
Application only in English. Missing large international markets. Vietnamese users would benefit from native language support.

## Requirements
1. Integrate i18n library (next-intl or react-i18next)
2. Extract all hardcoded strings to translation files
3. Support English, Vietnamese, Spanish, French initially
4. Implement language selector in UI
5. Store user language preference
6. Format dates/times per locale
7. Format numbers/currency per locale
8. Support RTL languages (preparation)
9. Translate all UI strings
10. Translate email templates
11. Translate error messages
12. Translate PDF CV templates
13. Handle pluralization rules per language
14. Test with native speakers
